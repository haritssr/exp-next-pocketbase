## Install package

pnpm install

## Download & install pocketbase on your computer

[pocketbase](https://pocketbase.io/docs/)

## Run pocket base server

./pocketbase serve

## Run your nextjs app locally

pnpm run dev

open localhost:3000 at browser URL
